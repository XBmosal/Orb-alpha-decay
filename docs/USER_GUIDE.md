# User Guide

**Status: Partial (shell only).** Flow Terminal is a **read-only** NQ/ES order-flow
viewer for personal market analysis and replay. It does not place orders.

## Current (Phase 0/1)
- Launch in **Mock** mode — a prominent **SIMULATED DATA** banner and a
  **READ-ONLY · NO ORDER ENTRY** indicator are always shown with synthetic data.
- The shell lays out the default workspace regions: top status bar (Instrument /
  Contract / Session / Status), main footprint/heatmap area, read-only DOM, CVD /
  profile, Time and Sales, and bottom tabs (Replay / Notes / Data / Diagnostics /
  Performance). These regions are labelled placeholders until their phases land.
- Display timezones: America/New_York (default), America/Chicago, UTC.
- Session templates: Full Globex, RTH, ETH, Overnight, Custom.
